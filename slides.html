Why limits?

Short answer: because derivatives (and integrals).

Derivatives: on a graph \(y=f(x)\text{,}\) what happens to \(\dfrac{\Delta y}{\Delta x}\) when \(\Delta x\) shrinks to zero?

Integrals: what happens when we add up more and more pieces that are getting smaller and smaller?

Limits let us talk about the value of a function at points where plugging in a number doesn’t make sense.

(For example, the value of \(\dfrac{\sin(x)}{x}\) when \(x=0\text{.}\))

Defining limits of functions (informally)

Limits provide us with a language for talking precisely about approximations.

The expression

\begin{equation*} \lim_{x\to c}f(x) = L \end{equation*}
means that when \(x\) is close to \(c\) (but not equal to \(c\)), we can expect \(f(x)\) to be close to \(L\text{.}\)

And the closer \(x\) gets to \(c\text{,}\) the closer \(f(x)\) gets to \(L\text{.}\)

More about that last (non) limit

We have language to talk about this situation.

The left-hand limit at \(c\) only considers values \(x\lt c\text{.}\)

The right-hand limit at \(c\) only considers values \(x\gt c\text{.}\)

Here, we would consider \(\lim\limits_{x\to 2^-}f(x)\) and \(\lim\limits_{x\to 2^+}f(x)\text{.}\)

Limit properties

The following properties of limits are true: \(\lim\limits_{x\to c}(k)=k\) for any constant \(k\text{,}\) \(\lim\limits_{x\to c}(x)=c\text{,}\) and if

\begin{equation*} \lim_{x\to c}f(x)=L\quad\text{ and } \quad \lim_{x\to c}g(x)=M\text{,} \end{equation*}
then:

  1. \(\displaystyle \lim\limits_{x\to c}(f(x)+g(x))=L+M\)

  2. \(\displaystyle \lim\limits_{x\to c}(f(x)-g(x))=L-M\)

  3. If \(k\) is a constant, then \(\lim\limits_{x\to c}(kf(x))=kL\text{.}\)

  4. \(\displaystyle \lim\limits_{x\to c}(f(x)g(x))=LM\)

  5. \(\lim\limits_{x\to c}\left(\frac{f(x)}{g(x)}\right)=\frac{L}{M}\text{,}\) if \(M\neq 0\text{.}\)

The key theorem

Theorem 1.

Let \(f(x)\) be defined for all \(x\) in an open interval containing \(c\text{,}\) except possibly not at \(x=c\text{.}\) Then

\begin{equation*} \lim_{x\to c}f(x) = L \end{equation*}
if and only if
\begin{equation*} \lim_{x\to c^-}f(x) = L \quad \text{ and } \quad \lim_{x\to c^+}f(x) = L\text{.} \end{equation*}

Continuity

Definition 2.

A function \(f\) is continuous at a point \(c\) if

\begin{equation*} \lim_{x\to c}f(x) = f(c)\text{.} \end{equation*}

\(f\) is continuous on an open interval \((a,b)\) if it is continuous at each point in the interval.

Buried in this equation:

  • \(f(x)\) is defined for all \(x\) close to \(c\)

  • \(\lim\limits_{x\to c}f(x)\) exists

    (So the left and right hand limits must be the same.)

  • \(f(c)\) is defined

  • The limit at \(c\) (which exists) must be equal to the value of \(f(c)\) (which is defined)

Properties of continuous functions

The following functions are continuous on their domains:

  • polynomial and rational functions

  • radical (root) functions

  • trig functions: \(\sin(x),\cos(x),\tan(x),\csc(x),\sec(x),\cot(x)\)

  • exponential functions: \(e^x, 2^x, 10^x\text{,}\) etc.

  • logarithms: \(\ln(x), \log_2(x), \log_{10}(x)\text{,}\) etc.

  • Any function formed from any of the above using addition, subtraction, multiplication, division, and composition. (As long as we don’t divide by 0.)

Note: some of these functions have vertical asymptotes. These are points of discontinuity, but they are not in the domain.

Types of discontinuity

There are many ways that a function can fail to be continuous at a point.

Some of these are common enough that they have names:

  1. A function \(f\) has a jump discontinuity at \(c\) if

    \begin{equation*} \lim_{x\to c^-}f(x) \neq \lim_{x\to c^+}f(x) \end{equation*}

  2. A function \(f\) has a removable discontinuity at \(c\) if \(\lim\limits_{x\to c}f(x)\) exists, but does not equal \(f(c)\)

  3. A function \(f\) has an infinite discontinuity at \(c\) if it has a vertical asymptote \(x=c\text{.}\)

Intermediate Value Theorem

Theorem 3.

Let \(f\) be a continuous function on an interval \([a,b]\text{,}\) and suppose that \(f(a)\lt f(b)\text{.}\)

Then for every \(y\) such that \(f(a)\lt y\lt f(b)\text{,}\) there is some \(x\in (a,b)\) such that \(f(x)=y\text{.}\)

Infinite limits

A function \(f\) is unbounded (or has an infinite limit) at \(c\) if the value of \(f(x)\) can be arbitrarily large when \(x\) is close to \(c\text{.}\)

Basic principle: if \(x\) is very small, \(\dfrac1x\) will be very big.

Here “large” could mean large and positive, or large and negative.

Notation includes:

  • \(\displaystyle \lim\limits_{x\to c}f(x)=\infty\)

  • \(\displaystyle \lim\limits_{x\to c^+}f(x)=-\infty\)

  • \(\displaystyle \lim\limits_{x\to c^-}f(x)=\infty\)

etc.

Infinite limits are associated with vertical asymptotes, and are usually caused by a division by zero.

Tangent lines

The tangent line at a point \((a,f(a))\) on a graph \(y=f(x)\) is the line with slope given by

\begin{equation*} m = f'(a) = \lim_{h\to 0}\frac{f(a+h)-f(a)}{h}\text{.} \end{equation*}

The slope \(f'(a)\) (read f prime at a) is called the derivative of \(f\) at \(a\text{.}\)

Notice that this is the same calculation we’d use to find instantaneous velocity.

Recap: derivative as slope of tangent line

Recall: the derivative of a function \(f\) is given by

\begin{equation*} f'(x) = \lim_{h\to 0}\frac{f(x+h)-f(x)}{h}\text{.} \end{equation*}

The definition was motivated by finding the slope of a tangent line. (We can also think of instantaneous velocity.)

At a point \((c,f(c))\) on a graph \(y=f(x)\text{,}\) the equation of the tangent line is

\begin{equation*} y = f(c)+f'(c)(x-c)\text{.} \end{equation*}

The derivative also gives a linear approximation: for small \(\Delta x\text{,}\)

\begin{equation*} \Delta y = f(c+\Delta x)-f(c) \approx f'(c)\Delta x\text{.} \end{equation*}

Inverse functions

Recall: a function \(f(x)\) is one-to-one if it never has the same \(y\) value for two different \(x\) values. That is, if \(x_1\neq x_2\text{,}\) then \(f(x_1)\neq f(x_2)\text{.}\)

This means that the equation \(y=f(x)\) has a unique solution for \(x\text{,}\) for every \(y\) in the range of \(f\text{.}\) We set \(x=f^{-1}(y)\text{,}\) and call \(f^{-1}\) the inverse of \(f\text{.}\)

Usually we swap \(x\) and \(y\) and write \(y=f^{-1}(x)\) (because we like functions to depend on \(x\)).

Note that \(y=f^{-1}(x)\) is the same as \(f(y)=x\text{.}\)

Also note \(f(f^{-1}(x))=f(y)=x\text{,}\) and \(f^{-1}(f(y))=f^{-1}(x)=y\text{,}\) so when we compose a function with its inverse, they cancel out.

Extreme Values

Definition 6.

Let \(f\) a function with domain \(D\text{.}\)

For a point \(c\in D\text{,}\) we call the value \(M=f(c)\) a global maximum (or absolute maximum) if \(f(c)\geq f(x)\) for all \(x\in D\text{.}\)

We call the value \(m=f(c)\) a global minimum (or absolute minimum) if \(f(c)\leq f(x)\) for all \(x\in D\text{.}\)

Note that the maximum and minimum are values of the function (\(y\) values).

It is possible for a max/min to occur at more than one \(x\) coordinate.

We use language like “\(f(2)=7\) is the absolute maximum”, or “\(f\) has a global maximum value of \(7\) when \(x=2\text{.}\)”

Extreme Value Theorem

Theorem 7. Extreme Value Theorem.

Let \(f(x)\) be a continuous function on a closed interval \([a,b]\text{.}\)

Then there exist points \(c_1,c_2\in [a,b]\) such that \(M=f(c_1)\) is the global maximum of \(f\) on \([a,b]\text{,}\) and \(m=f(c_2)\) is the global minimum of \(f\) on \((a,b)\text{.}\)

Question: where in the interval can the points \(c_1,c_2\) occur? What can we say about the behaviour of \(f(x)\) at these points?

Local maxima and minima

A value \(M=f(c)\) is a local maximum (or relative maximum) if \(f(c)\geq f(x)\) for all \(x\) “near” \(c\text{.}\)

For our purposes, “near” will usually mean “in an open interval containing \(c\)”.

A careful reading of the definition used in our textbook suggests that an “endpoint” maximum counts as a local maximum.

A local minimum is defined similarly, with \(f(c)\leq f(x)\text{.}\)

Finding extreme values

Suppose we have a continuous function \(f\) on a closed interval \([a,b]\text{.}\)

The EVT guarantees that global maximum and minumum values exist. How do we find them?

Observation 1: any global max or min is also a local max or min.

Observation 2: any local max or min occurs at one of three types of location:

  1. One of the two endpoints of the interval

  2. A point \(c\) where \(f'(c)=0\)

  3. A point \(c\) where \(f'(c)\) does not exist.

A point \(c\) where either \(f'(c)=0\) or \(f'(c)\) is undefined is called a critical point of \(f\text{.}\)

An algorithm

To summarize: to find the absolute maximum and minimum values of a continuous function \(f(x)\) on \([a,b]\text{:}\)

  1. Evaluate \(f\) at the endpoints: find \(f(a)\) and \(f(b)\)

  2. Compute \(f'(x)\text{,}\) and find all critical points of \(f\)

  3. Evaluate \(f\) at each critical point

  4. Compare the values found in steps 1 and 3. The largest is the global max, and the smallest is the global min.

Concavity

The “concavity of a graph” has to do with its curvature: in what way does the graph deviate from a straight line?

  • If the graph of a function \(f\) curves upward, we say it is concave up.

    • When the graph of \(f\) is concave up, secant lines lie above the graph, and tangent lines (where they exist) lie below the graph.

    • If \(f''(x)\gt 0\) on an interval \(I\text{,}\) the graph of \(f\) is concave up on \(I\)

  • If the graph of a function \(f\) curves downward, we say it is concave down.

    • When the graph of \(f\) is concave down, secant lines lie below the graph, and tangent lines (where they exist) lie above the graph.

    • If \(f''(x)\lt 0\) on an interval \(I\text{,}\) the graph of \(f\) is concave down on \(I\)

A point \(c\) where the graph of \(f\) changes concavity is called an inflection point.

And now for something new: tangent lines

Recall: given a graph \(y=f(x)\) of a differentiable function \(f\text{,}\) the tangent line to the graph at a point \((c,f(c))\) is

\begin{equation*} \ell(x) = f(c)+f'(c)(x-c)\text{.} \end{equation*}

Differentials

The point of the tangent line (AKA linearization) is twofold:

  1. It’s linear, and therefore simple.

  2. As long as \(\Delta x = x-c\) is small, \(\ell(x)\) is a good approximation of \(f(x)\text{.}\)

If \(x = c + \Delta x\text{,}\) and \(y=f(x)\text{,}\) we set:

\begin{align*} \Delta y \amp = f(x)-f(c) = f(c+\Delta x)-f(c)\\ dy \amp = \ell(x)-\ell(c) = f'(c)\Delta x \end{align*}

Note that if \(f(x)=x\text{,}\) we get \(y=x\text{,}\) and \(dx = f'(c)\Delta x = \Delta x\text{,}\) since \(f'(x)=1\text{.}\)

This means we can use \(dx\) and \(\Delta x\) interchangeably.

But the same is not true of \(dy\) and \(\Delta y\text{!}\) However, we can say that \(\Delta y \approx dy\text{,}\) if \(\Delta x\) is small.

Improving the linear approximation

When a linear approximation isn’t good enough, we can consider adding in higher-order corrections.

If the graph \(y=f(x)\) is concave down at \(c\text{,}\) will the linear approximation \(\ell(x)\) at \(c\) give an overestimate, or an underestimate?

What if the graph is concave up?

To account for this, we can add a “second-order correction”. That is, we can look for a quadratic approximation

\begin{equation*} q(x) = A+B(x-c)+C(x-c)^2\text{.} \end{equation*}

We want \(q(c)=f(c)\text{,}\) so \(A=f(c)\text{.}\)

We want \(q'(c)=f'(c)\) (it seemed like a good idea for the linear approximation), so \(B = f'(c)\text{.}\) (Notice that \(q'(x)=B+2C(x-c)\text{.}\))

It seems reasonable for the quadratic approximation to agree with the concavity of \(f(x)\text{,}\) so we ask that \(q''(c)=f''(c)\text{,}\) which implies \(2C=f''(c)\text{,}\) or \(C = \frac{f''(c)}{2}\text{.}\)

A quadratic approximation

We just found that a quadratic approximation to \(f(x)\) at \(c\) is

\begin{equation*} q(x) = f(c)+f'(c)(x-c)+\frac{f''(c)}{2}(x-c)^2\text{.} \end{equation*}

What is the quadratic approximation to \(f(x)=\sqrt{x}\) at \(4\text{?}\)

Recall that we already found the linear approximation \(\ell(x)=2+\frac14(x-4)\text{.}\)

  1. \(\displaystyle q(x)=-\dfrac14 x^{-3/2}\)

  2. \(\displaystyle q(x) = -\dfrac{1}{64}(x-4)^2\)

  3. \(\displaystyle q(x) = 2+\dfrac14(x-4)-\dfrac{1}{64}(x-4)^2\)

  4. \(\displaystyle q(x) = 2+\dfrac14-\dfrac{1}{64}\)

Taylor polynomials

The continuation of this process of adding higher-order corrections leads to what are called Taylor polynomials

Given a function \(f(x)\) such that derivatives of \(f\) of all orders exist at \(c\text{,}\) the degree \(n\) Taylor polynomial of \(f\) at \(c\) is

\begin{equation*} p_n(x) = f(c)+f'(c)(x-c)+\frac{f''(c)}{2!}(x-c)^2+\frac{f'''(c)}{3!}(x-c)^3+\cdots + \frac{f^{(n)}(c)}{n!}(x-c)^n\text{.} \end{equation*}
The special case when \(c=0\) is called a Maclaurin polynomial.

Note that \(f^{(n)}(c)\) means the \(n\)th derivative of \(f\) at \(c\text{,}\) and \(n!\) is the factorial of \(n\text{:}\) the product of all integers from 1 to \(n\text{.}\)

Note \(1!=1\text{,}\) \(2! = 1(2)=2\text{,}\) \(3! = 1(2)(3)=6\text{,}\) \(4! = 1(2)(3)(4)=24\text{,}\) and so on.

Taylor polynomials can be a very effective tool for approximating a function.

Riemann sums

The sums we create by adding up areas of rectangles are called Riemann sums, after Bernhard Riemann.

The general process is as follows. Given \(f(x)\) on \([a,b]\text{:}\)

  1. Divide the interval \([a,b]\) into smaller sub-intervals by choosing points \(a=x_0\lt x_1\lt x_2\lt\cdots\lt x_n=b\) (this is called a partition, which we’ll label \(P\))

  2. The length of the interval \([x_{i-1},x_i]\) is denoted \(\Delta x_i = x_i-x_{i-1}\)

  3. It is usually convenient to make each sub-interval the same length, in which case \(\Delta x_i = \dfrac{b-a}{n}\) (a uniform partition)

  4. In each interval, choose a point \(c_i\in [x_{i-1},x_i]\text{,}\) and plug this into your function to get \(f(c_i)\)

  5. The area of the \(i\)th rectangle is then \(A_i = f(c_i)\Delta x_i\text{,}\) for \(i=1,2,\ldots, n\)

  6. Adding the areas gives the Riemann sum \(\di R(f,P) = \sum_{i=1}^nf(c_i)\Delta x_i\)

Common choices for \(c_i\) are the left endpoint (\(c_i=x_{i-1}\)), the right endpoint (\(c_i=x_i\)), or the midpoint (\(c_i =(x_{i-1}+x_i)/2\)).

Take it to the limit

One more time: a Riemann sum lets us approximate a definite integral using rectangles.

And the more rectangles we use, the better our approximation gets.

What if we let \(n\to\infty\text{?}\)

Given \(y=f(x)\text{,}\) \(a\leq x\leq b\text{,}\) we set \(\Delta x = \dfrac{b-a}{n}\text{,}\) and then:

\begin{equation*} x_0 = a, x_1 = a+\Delta x, x_2 = a+2\Delta x,\cdots,x_i = a+i\Delta x,\cdots, x_n = a+n\Delta x = b \end{equation*}

In the limit, it turns out to not matter how we choose the \(c_i\text{,}\) so choose \(c_i=x_i\) for convenience.

Then: \(\di \int_a^b f(x) \,dx = \lim_{n\to\infty}\sum_{i=1}^n f(x_i)\Delta x\)

There must be a better way!

Didn’t we mention last time that definite integrals are somehow related to antiderivatives?

We did! If we know the velocity function \(v(t)\text{,}\) we can recover position, up to a constant:

\begin{equation*} x(t) = \int v(t)\,dt = F(t)+C\text{,} \end{equation*}
for some antiderivative \(F\) of \(v\text{,}\) and some constant \(C\text{.}\)

Even if we don’t know \(C\text{,}\) we can still find the change in position, between time \(t=a\) and time \(t=b\text{:}\)

\begin{equation*} x(b)-x(a) = (F(b)+C)-(F(a)+C) = F(b)-F(a)\text{.} \end{equation*}

But \(x(b)-x(a)\) is also the area under \(y=v(t)\text{,}\) for \(a\leq t\leq b\text{:}\)

\begin{equation*} \int_a^b v(t)\,dt = x(b)-x(a)\text{,} \end{equation*}
where \(x'(t)=v(t)\text{.}\)

Fundamental Theorem of Calculus

Technically, this is FTC part 2, even though we’re doing it first.

Theorem 13.

Let \(f\) be a continuous function defined on the interval \([a,b]\text{.}\)

Let \(F\) be any antiderivative of \(f\) on \([a,b]\text{.}\)

Then \(\di \int_a^b f(x)\,dx = F(b)-F(a)\text{.}\)

Notation note: we often write \(F(x)\Bigr\rvert_a^b = F(b)-F(a)\) as an intermediate step when evaluating a definite integral. First we get the antiderivative, and then we plug in the endpoints.

But why does it work?

Let \(f\) be continuous on \([a,b]\text{.}\)

Let \(F\) be an antiderivative of \(f\) on \([a,b]\text{,}\) and let \(a=x_0\lt x_1\lt x_2\lt\ldots\lt x_n\) be a partition of \([a,b]\text{.}\)

On each subinterval \([x_{i-1},x_i]\text{,}\) \(F(x)\) is continuous, and differentiable.

By the Mean Value Theorem, there is some \(c_i\in [x_{i-1},x_i]\) such that \(F(x_i)-F(x_{i-1})=F'(c_i)(x_i-x_{i-1})\text{.}\)

Now notice that \(F'(c_i)(x_i-x_{i-1}) = f(c_i)\Delta x_i\text{.}\)

So, \(\di \sum_{i=1}^n f(c_i)\Delta x_i = \sum_{i=1}^n (F(x_{i})-F(x_{i-1})) = F(b)-F(a)\text{.}\)

Some other considerations

Observations of \(symmetry\) can often be useful.

A function \(f\) is even if \(f(-x)=f(x)\) for each \(x\) in its domain.

For even functions \(\di \int_{-a}^a f(x)\,dx = 2\int_0^a f(x)\,dx\text{.}\)

A function \(f\) is odd if \(f(-x)=-f(x)\) for each \(x\) in its domain.

For odd functions, \(\di \int_{-a}^a f(x)\,dx = 0\text{.}\)

Substitution

If \(F(x)=f(g(x))\text{,}\) then \(F'(x)=f'(g(x))g'(x)\text{.}\) Therefore, \(\di \int f'(g(x))g'(x)\,dx = f(g(x))+C\text{.}\)

If we write \(u=g(x)\text{,}\) then \(du = g'(x)\,dx\text{,}\) so \(f'(g(x))g'(x)\,dx = f'(u)\,du\text{.}\)

We call this \(u\)-substitution.

Example: for the integral \(\di\int 4x(x^2+4)^7\,dx\text{,}\) we could multiply everything out.

Or... let \(u=x^2+4\text{.}\) Then \(du = 2x\,dx\) (so \(4x\,dx = 2\,du\)), and \(\displaystyle \int 4x(x^2+4)^7\,dx = \int 2u^7\,du \text{.}\)

Definite integrals

What happens when we use substitution for a definite integral? There are two options:

  1. Use substitution on the corresponding indefinite integral, then use the FTC.

  2. Adjust the bounds on your integral as well as the integrand.

Example: evaluate \(\di \int_0^3 \frac{4x}{x^2+1}\,dx\text{.}\)

  • Option 1: consider \(\int \frac{4x}{x^2+1}\,dx\) first. Let \(u=x^2+1\text{.}\) Then \(du=2x\,dx\) and \(\int \frac{4x}{x^2+1}\,dx = \int\frac{2}{u}\,du = 2\ln\lvert u\rvert +C=2\ln(x^2+1)+C\)

  • Using the FTC, \(\int_0^3 \frac{4x}{x^2+1}\,dx = 2\ln(x^2+1)\bigr\vert_0^3 = 2\ln(10)\text{.}\)

  • Option 2: adjust bounds. If \(u=x^2+1\text{,}\) then when \(x=0\text{,}\) \(u=0^2+1=1\text{,}\) and when \(x=3\text{,}\) \(u=3^2+1=10\text{.}\)

  • Then \(\int_0^3\frac{4x}{x^2+1}\,dx = \int_1^{10}\frac{2}{u}\,du = 2\ln\lvert u\rvert\bigr\vert_1^{10} = 2\ln(10)\text{.}\)
